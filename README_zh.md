# CodeActor Agent

еҹәдәҺ Go иҜӯиЁҖејҖеҸ‘зҡ„ **Hub-and-SpokeпјҲдёӯжһў-иҫҗжқЎпјүеӨҡжҷәиғҪдҪ“жһ¶жһ„** AI иҮӘдё»зј–зЁӢеҠ©жүӢгҖӮ

CodeActor Agent еҚҸи°ғеӨҡдёӘдё“з”ЁжҷәиғҪдҪ“вҖ”вҖ”жҢҮжҢҘе®¶пјҲConductorпјүгҖҒд»“еә“еҲҶжһҗе‘ҳпјҲRepo-AnalystпјүгҖҒзј–з Ғе·ҘзЁӢеёҲпјҲCoding-Engineerпјүе’ҢеҜ№иҜқеҠ©жүӢпјҲChat-AssistantпјүвҖ”вҖ”иҮӘдё»е®ҢжҲҗеӨҚжқӮзҡ„иҪҜд»¶е·ҘзЁӢд»»еҠЎпјҢе…·еӨҮиҮӘжҲ‘зә й”ҷиғҪеҠӣгҖӮ

## зү№жҖ§

- **еӨҡжҷәиғҪдҪ“жһ¶жһ„** вҖ” дёӯеӨ® Conductor е°Ҷд»»еҠЎе§”жҙҫз»ҷдё“з”ЁеӯҗжҷәиғҪдҪ“пјҲд»“еә“еҲҶжһҗгҖҒд»Јз Ғзј–иҫ‘гҖҒйҖҡз”ЁеҜ№иҜқпјү
- **дё°еҜҢзҡ„е·Ҙе…·зі»з»ҹ** вҖ” 14 дёӘеҶ…зҪ®е·Ҙе…·пјҢж¶өзӣ–ж–Үд»¶ж“ҚдҪңгҖҒд»Јз ҒжҗңзҙўгҖҒиҜӯд№үеҲҶжһҗгҖҒShell жү§иЎҢе’Ңи®ӨзҹҘеҸҚжҖқ
- **иҮӘжҲ‘дҝ®жӯЈ** вҖ” `thinking` е·Ҙе…·дҪҝ Agent иғҪеӨҹеңЁеҮәй”ҷж—¶еҲҶжһҗеҺҹеӣ е№¶жҒўеӨҚпјҢйҒҝе…ҚзӣІзӣ®йҮҚиҜ•
- **еҸҢдәӨдә’жЁЎејҸ** вҖ” TUI з»Ҳз«Ҝз•Ңйқўз”ЁдәҺжң¬ең°дҪҝз”ЁпјӣHTTP + WebSocket жңҚеҠЎз”ЁдәҺ IDE/Web йӣҶжҲҗ
- **еӨҡжҸҗдҫӣе•Ҷ LLM ж”ҜжҢҒ** вҖ” е°Ҹзұі MiMoгҖҒйҳҝйҮҢ QwenгҖҒDeepSeekгҖҒMistralгҖҒAWS Bedrock зӯүпјҢйҖҡиҝҮ OpenAI е…је®№ API
- **жөҒејҸиҫ“еҮә** вҖ” AI еӣһеӨҚгҖҒе·Ҙе…·и°ғз”Ёе’Ңз»“жһңе®һж—¶жөҒејҸдј иҫ“
- **еҜ№иҜқи®°еҝҶ** вҖ” е®Ңж•ҙеҜ№иҜқдёҠдёӢж–ҮпјҲеҗ«е·Ҙе…·и°ғз”ЁеҺҶеҸІпјүпјҢи·ЁдјҡиҜқжҢҒд№…еҢ–
- **д»“еә“еҲҶжһҗ** вҖ” иҮӘеҠЁд»Јз Ғеә“и°ғжҹҘпјҢеҢ…еҗ«зӣ®еҪ•ж ‘гҖҒи°ғз”Ёеӣҫе’ҢиҜӯд№үжҗңзҙў

## еҝ«йҖҹејҖе§Ӣ

### зҺҜеўғиҰҒжұӮ

- Go 1.23+
- `ripgrep` (`rg`) вҖ” е…Ёж–ҮжӯЈеҲҷжҗңзҙў
- `fzf` вҖ” жЁЎзіҠж–Үд»¶жҗңзҙўпјҲеҸҜйҖүпјү
- иҝҗиЎҢдёӯзҡ„ `codeactor-codebase` жңҚеҠЎпјҲжҲ–и®ҫзҪ® `CODEBASE_URL`пјү

### е®үиЈ…

```bash
git clone https://github.com/your-org/codeactor-agent.git
cd codeactor-agent
go build -o codeactor .
```

### й…ҚзҪ®

еҲӣе»ә `$HOME/.codeactor/config/config.toml`пјҡ

```toml
[http]
server_port = 9080

[llm]
use_provider = "siliconflow"

[llm.providers.siliconflow]
model = "deepseek-ai/DeepSeek-V3.2"
temperature = 0.0
max_tokens = 23000
api_base_url = "https://api.siliconflow.cn/v1"
api_key = "your-api-key-here"

[app]
enable_streaming = true

[agent]
conductor_max_steps = 30
coding_max_steps = 50
repo_max_steps = 30
lang = "Chinese"
```

### иҝҗиЎҢ

**TUI жЁЎејҸ**пјҲз»Ҳз«Ҝз•Ңйқўпјүпјҡ
```bash
./codeactor tui
# жҲ–жҗәеёҰд»»еҠЎж–Үд»¶пјҡ
./codeactor tui --taskfile TASK.md
```

**HTTP жңҚеҠЎжЁЎејҸ**пјҲAPI + WebSocketпјүпјҡ
```bash
./codeactor http
# жңҚеҠЎеҗҜеҠЁеңЁ http://localhost:9080
```
