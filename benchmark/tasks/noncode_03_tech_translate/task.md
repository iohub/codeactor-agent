# 任务：技术规范文档翻译

请将项目中的英文技术规范 `SPEC.md` 完整翻译为中文，保存为 `SPEC_CN.md`。

## 翻译要求

1. **保持结构完整** — 保留原文的所有章节标题、编号和层级
2. **术语准确** — 技术术语翻译符合行业标准：
   - Rate Limiter → 限流器
   - Circuit Breaker → 熔断器
   - API Gateway → API 网关
   - Token Bucket → 令牌桶
   - Round-robin → 轮询
   - Health Check → 健康检查
   - Prometheus → Prometheus（保留原名）
3. **首次标注** — 专业术语首次出现时在括号中保留英文原文，如"限流器（Rate Limiter）"
4. **自然流畅** — 译文应符合中文技术文档的表达习惯，避免生硬的直译
5. **表格保留** — 原文中的表格保持格式，表头和内容均需翻译
6. **技术准确** — 不改变任何技术细节和参数值

## 验收标准

生成的 `SPEC_CN.md` 需完整、准确翻译原文内容。
